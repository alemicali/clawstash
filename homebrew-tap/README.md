# homebrew-clawstash

Homebrew tap for [clawstash](https://github.com/alemicali/clawstash).

## Install

```sh
brew tap alemicali/clawstash
brew install clawstash
```

Then run the setup wizard:

```sh
clawstash setup
```

## About

This tap will be published as a separate repo at `github.com/clawstash/homebrew-clawstash`.
For now it lives inside the main clawstash repo as a reference.
